📌 LostFoundry

 LostFoundry is a web-based application that helps users report and recover lost items within a community. Users can post details of lost or found items with an optional image, making it easier to identify and return belongings.

🧠 Problem Statement

 How can we simplify the process of reporting and recovering lost items in a college environment efficiently?

📊 System Details

 Users can submit item details along with descriptions and optional images. Each post includes contact information so others can reach out directly. An admin manages posts and removes them once items are found.

🔧 Technologies Used

 HTML
 CSS
 Firebase
 Cloudinary

⚙️ Features Implemented

 Post lost and found items
 Optional image upload
 Display items on the platform
 Contact via email
 Admin panel to manage and remove posts

🔄 Workflow

 User submits item details
 Data is stored in Firebase
 Images are stored in Cloudinary
 Posts are displayed on the platform
 Admin updates or removes resolved posts

✅ Results

 Successfully created a working platform for managing lost and found items
 Enabled direct communication between users
 Maintained updated records through admin control
